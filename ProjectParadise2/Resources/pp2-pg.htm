Project Paradise 2 – Guia de Instalação

Início rápido e ajuda

Geral

Antes de iniciar a instalação, observe as seguintes instruções:

  • Versões do Windows suportadas: Windows 7, 8, 10, 11
  • Memória RAM: pelo menos 4 GB
  • Espaço livre em disco: pelo menos 18 GB
  • Placas de vídeo recomendadas: NVIDIA GeForce 8800 GTX / ATI Radeon HD 3870 ou superior (mínimo de 512 MB VRAM)
  • Processador: Intel Core 2 Duo 2,2 GHz ou AMD Athlon X2 4400+
  • Microsoft Visual C++ Runtime 2005/2008 (32-Bit)

O jogo deve estar instalado de forma limpa. Por favor, não use versões de jogo repacked, manipuladas ou alteradas de qualquer outra forma.

O Project Paradise 2 não altera os arquivos do seu jogo por padrão. As alterações ocorrem apenas quando solicitadas explicitamente pelo usuário (por exemplo, através de configurações de perfil ou atualizações).

O Project Paradise 2 não fornece cópias de jogos, downloads de jogos ou qualquer conteúdo protegido por direitos autorais. Para jogar, é necessária uma versão legal e própria do respectivo jogo.

O Launcher não distribui vírus, malware ou spyware. O código completo do launcher é público no GitHub e pode ser verificado a qualquer momento.

O uso do launcher é de sua inteira responsabilidade. Não assumimos nenhuma responsabilidade por perda de dados, danos ao jogo, problemas no sistema ou outros danos decorrentes do uso do launcher ou de funções associadas.

Ao usar o Project Paradise 2, você concorda com estes termos.

Configuração do Launcher

  • Primeira inicialização do Launcher

    • No lado esquerdo, você encontrará o menu de navegação. No canto inferior direito, ao lado dos jogos, você verá os perfis de jogo criados.

    Se você já jogou o jogo anteriormente, será perguntado se deseja que um perfil seja criado automaticamente.

    Se selecionar "Sim" (Ja), o launcher tentará fazer isso por você. Se "Não" (Nein), você mesmo deverá criar o perfil. Para isso, clique em "Profile".

    Clique em "Create New Profile", digite um nome para o perfil e clique em "Select Game". Em seguida, selecione o executável (.exe) do jogo. O launcher suporta TDU1 e TDU2. Assim que um jogo for selecionado, mais opções estarão disponíveis.

    As configurações:

    1. Magic RAM → Permite que o jogo use mais RAM dinamicamente.
    2. Magic Cores → Tenta atribuir mais núcleos de CPU ao jogo.
    3. Magic Prio → Define a prioridade do processo do jogo como superior ao normal.
    4. Vehicle Dirt → Ativa ou desativa o acúmulo de sujeira nos veículos.
    5. Vehicle Damage → Ativa ou desativa danos nos veículos.
    6. Online Mode → Inicia o jogo no modo online ou offline.
    7. Start Args → Parâmetros de inicialização passados ao jogo (ex: no TDU1: -w para modo janela).

    Observação: Os interruptores (Toggle) funcionam como DESLIGADO à esquerda e LIGADO à direita.

    Salve o perfil. Dependendo da versão do jogo, você será informado se uma atualização é necessária.

    Se receber a mensagem "Is that okay?", clique novamente em "Profile" e depois em "Edit" para editar a instância do perfil atual.

    Agora, novos botões estão disponíveis. Primeiro, clique em "Spiel-Update" (Atualizar Jogo) e deixe-o rodar completamente. Isso atualizará seu jogo.

    *IMPORTANTE* Para jogar online, você precisa da versão mais recente (marcada em verde no perfil). Perfis de jogo vermelhos não são compatíveis com o modo online!

    Assim que tudo estiver pronto, clique em "Start", selecione o perfil criado e clique em "Spielen" (Jogar).

    Conexão / Conectividade do Lobby

    Clique em "Informationen".

    Aqui você encontrará informações sobre seu hardware e conexão de rede. Um ponto importante é a aba de rede (Network):

    1. State → Resultado do teste de rede (Aberto, Moderado, Bloqueado). Infelizmente, este teste nem sempre é confiável.
    2. Server → O launcher estabelece uma conexão com o servidor. Indica se o servidor está acessível.
    3. Player → Se a conexão for bem-sucedida, o servidor tentará dar um "ping" no launcher com um pequeno pacote. Se for recebido, a conexão está aberta.

    Para poder hospedar lobbies, geralmente é necessário um status "Aberto" ou "Player acessível". No entanto, pode funcionar sem isso se você já esteve em um lobby com essa pessoa antes.

    O launcher suporta UPnP/PnP. Isso significa que ele tenta configurar automaticamente o redirecionamento de portas para você. Para isso, a opção deve estar ativada (encontrada nas configurações).

Opcional

Tentamos tornar tudo o mais simples possível para o usuário.

Para isso, existe a aba "Services" (Serviços).

Aqui você encontrará configurações opcionais para facilitar o manuseio de Savegames.

As opções incluem a função de nuvem (Cloud) e a função de backup local.

Função de backup de savegame local:

Pontos de salvamento podem ser criados automaticamente pelo launcher.

Existem duas opções: antes e depois do início do jogo. Isso significa que um ponto de backup é criado antes do início do jogo ou ao fechá-lo.

"Speicherpunkte" é o número máximo de backups que o launcher pode criar. Quando esse número é atingido, o backup mais antigo é sempre substituído (recomendado: 4).

Se você receber uma mensagem no jogo sobre um savegame corrompido, não continue. Vá em Services e abra a pasta de backup (Backup-Ordner).

Abra um backup antigo e substitua o seu savegame atual. Se estiver intacto, o jogo continuará do último ponto de salvamento.

Savegame na Nuvem (Cloud)

Disponibilizamos espaço de armazenamento gratuito para os usuários fazerem o upload de seus savegames. Um perfil online é necessário para isso.

Você deve fazer login com seus dados. Atenção: O launcher não faz upload automático de savegames, pois não podemos verificar se eles estão corrompidos.

Portanto, o upload deve ser feito manualmente de tempos em tempos.

Ajuda / Solução de Problemas

  • O launcher tem uma atualização, mas o atualizador não faz nada → Abra a pasta do jogo e inicie o Updater manualmente.
  • Uso de um controle de PlayStation → É necessário um software de emulação que emule o controle como um controle de Xbox (ex: DS4Windows).
  • Alternar o jogo para o modo tela cheia → Pressione ALT + ENTER.
  • Timeout da placa de vídeo AMD → Infelizmente, os drivers AMD a partir de março de 2025 apresentam problemas. Fazer um downgrade para o driver de fevereiro (se disponível) pode ajudar.
  • Sem som no jogo → Verifique o dispositivo de áudio padrão do Windows. Fones de ouvido e alto-falantes Bluetooth não são suportados pelo jogo.
  • Vejo jogadores no mapa, mas não no jogo → Isso pode acontecer se o servidor não conseguir estabelecer uma conexão (sem jogadores hospedáveis).
  • Meu jogo modificado trava → Recomendamos o uso de um jogo original. Em jogos modificados pelo próprio usuário, nosso suporte é limitado.
  • Onde encontro os logs do launcher → Eles estão na pasta Project Paradise 2 em "Meus Documentos".
  • Onde encontro o savegame → Em "Meus Documentos", na pasta Eden Games.
  • Erro na ativação do jogo: Ferramenta de ativação: Baixar aqui
  • Telas pretas, travamentos: Placas Nvidia → Vá em "Profile" e edite o perfil desejado. Lá você encontrará o "Mod Browser" com dois fixes de GPU/Tela preta (DxVoodoo recomendado, Vulkan-fix para placas TI). Instale um fix e veja se a situação melhora.
  • "Conexão com o servidor falhou": Atualize o jogo, verifique as portas e aplique o patch de conexão via launcher. Para isso, clique em "Profile", "Edit" e "Patch Connection".
  • Outros mods multiplayer: Instalação limpa recomendada, não atualize EXEs de repacks.
