1. Requirements

Before proceeding with the installation, ensure that your system meets the following requirements:

  • Windows 7, Windows 8, Windows 10, Windows 11
  • At least 4 GB RAM
  • At least 18 GB of free hard disk space
  • Graphics: NVIDIA GeForce 8800 GTX/ATI Radeon HD 3870 or faster, at least 512 MB. Newer GPUs (mainly Nvidia) might cause issues with the game and require additional software fixes.
  • Processor: Intel Core 2 Duo 2.2GHz or AMD Athlon X2 4400+
  • Microsoft C++ Runtime 2005/2008 (32-bit version)

It's also important that the game is already installed. Use clean versions only. Repacked or pre-modified games can cause issues!

2. Installation

Follow the steps below to install the project:

  1. Open the launcher settings and click "Select Game". If the original launcher was previously opened, it will automatically show the correct game directory. Select the .exe file there. If not, manually navigate to the game directory and select the .exe file.
  2. The launcher will always attempt to start the selected game, regardless of the launcher's location. If a message appears stating that the game is not up-to-date, perform a "Game Check". This verifies and updates the existing files.
  3. Wait until the process is completed. If the download is slow, we recommend using a VPN for a better connection to the server location (Berlin, Germany).
  4. If you own the game on Steam, there are two important points:
    • If the launcher is in the game directory, add the launch parameter -direct.
    • Enable the Steam option (toggle) in the launcher in both cases.
  5. No registration is required for Project Paradise 2. You create an account directly in the game. Use memorable information and avoid using an email address whenever possible. We aim to collect as little personal data as possible.
  6. Before creating an account, check if the desired name is already taken. You can use the function on our website in the login section for this.
  7. That's it for the installation! If you receive an error stating that no connection could be established, click "Set Connection". The game's connection file will be automatically adjusted.

3. Troubleshooting

  • Error during game activation: This often happens with public game keys. There is a tool for generating an activation code:
    You can find the tool here: We are not affiliated with the creator of the tool. If you encounter problems, please contact them directly.
  • Black screens on maps, crashes with game maps, crashes when creating screenshots:
    This issue often occurs with Nvidia graphics cards. We have provided three fixes in the launcher under "Discovery." Depending on your GPU and system, one of these solutions may work better for you.
    Important: DxVoodoo has helped most users with RTX cards. It includes a configuration tool, which can be found in the settings folder of the launcher. Adjust the settings as needed. Based on experience, increasing memory allocation and setting the DirectX level to maximum often helps.
    The Vulkan crash fix works better with TI cards and requires no additional settings. Just download it, and the launcher will handle the rest.
    Using these fixes won’t solve 100% of crashes but will reduce their frequency. We also recommend using the 4GB patched EXE available in the Discovery tab.
  • "Failed to connect to server" error in the game:
    This error can have various causes. Ensure that a) the game is up to date and b) your connection is not blocked (e.g., incorrect port forwarding). To be safe, set up the connection via the launcher.
    If you use repacked games, it’s possible that the game’s exe is rejected online. To avoid issues, support for such exe files has been disabled.
    Is your username already taken? Visit the website: Check here if your desired name is available. If the name is already taken, use a different one or contact a team member (Discord, forum).
  • Other multiplayer mod installation:
    If you came here via another mod project, we recommend a clean installation. The game file updater can also help with installation.
    The mods you received from the mod are 1:1 compatible. Important: Never run the game file updater if you are using a repacked game, as this can damage the installation!

4. Frequently Asked Questions (FAQ)

  • Is money cheating allowed? Basically, we don’t care, but you should know that we have implemented an anti-cheat system that reacts to manipulated regions in memory and terminates the game.
  • Can I use mod cars? Yes, as long as they are not a) used in the Community Racing Center for official races or b) disruptive to other players (causing crashes, etc.).
  • Is multiplayer free? Yes, there is no fee, donation, or similar required for online access. If you’d like to help with server costs (of course, only if you can), you can do so via Patreon.
