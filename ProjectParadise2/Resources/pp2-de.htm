1. Voraussetzungen

Bevor Sie mit der Installation fortfahren, stellen Sie sicher, dass Ihr System die folgenden Anforderungen erfüllt:

  • Windows Windows 7, Windows 8, Windows 10, Windows 11
  • Mindestens 4 GB RAM
  • Mindestens 18 GB freien Festplattenspeicher
  • Grafik: NVIDIA GeForce 8800 GTX/ ATI Radeon HD 3870 oder schneller, mindestens 512 MB. Neuere GPUs (hauptsächlich Nvidia) könnten Probleme mit dem Spiel verursachen und müssen mit zusätzlicher Software behoben werden.
  • Prozessor: Intel Core 2 Duo 2.2GHz oder AMD Athlon X2 4400+
  • Microsoft C++ Runtime 2005/2008 als 32-bit Version

Es ist au√üerdem wichtig, dass das Spiel bereits installiert ist. Aus Erfahrung sollten nur saubere Versionen verwendet werden. Repacked oder bereits modifizierte Spiele können Probleme verursachen!

2. Installation

Folgen Sie den unten stehenden Schritten, um das Projekt zu installieren:

  1. Öffnen Sie die Einstellungen des Launchers und klicken Sie auf "Select Game". Wenn der Original-Launcher zuvor geöffnet war, wird automatisch das richtige Spielverzeichnis angezeigt. Wählen Sie dort die .exe-Datei aus. Falls dies nicht der Fall ist, navigieren Sie manuell in das Spielverzeichnis und wählen Sie die .exe-Datei aus.
  2. Der Launcher versucht immer, das ausgewählte Spiel zu starten, unabhängig davon, wo sich der Launcher befindet. Wenn die Meldung erscheint, dass das Spiel nicht auf dem neuesten Stand ist, führen Sie einen "Spiel Check" aus. Dieser prüft die vorhandenen Dateien und aktualisiert sie bei Bedarf.
  3. Warten Sie, bis die Meldung erscheint, dass der Vorgang abgeschlossen ist. Sollte der Download langsam sein, empfehlen wir die Verwendung eines VPNs, um eine bessere Verbindung zum Serverstandort (Berlin, Deutschland) zu erhalten.
  4. Falls Sie das Spiel über Steam besitzen, gibt es zwei wichtige Punkte:
    • Wenn sich der Launcher im Spielverzeichnis befindet, fügen Sie den Startparameter -direct hinzu.
    • Aktivieren Sie in beiden Fällen die Steam-Option (Toggle) im Launcher.
  5. Bei Project Paradise 2 ist keine Registrierung erforderlich. Sie erstellen einen Account direkt im Spiel. Bitte verwenden Sie Informationen, die Sie sich leicht merken können, und vermeiden Sie nach Möglichkeit die Verwendung einer E-Mail-Adresse. Wir möchten so wenige persönliche Daten wie möglich erfassen.
  6. Wir empfehlen, vor der Account-Erstellung zu prüfen, ob der gewünschte Name bereits vergeben ist. Dazu können Sie die Funktion auf unserer Website im Login-Bereich nutzen.
  7. Das war's mit der Installation! Wenn Sie eine Fehlermeldung erhalten, dass keine Verbindung hergestellt werden konnte, klicken Sie auf "Set Connection". Die Verbindungsdatei des Spiels wird automatisch angepasst.

3. Fehlerbehebung

  • Fehler bei der Aktivierung des Spiels: Dies passiert oft bei öffentlichen Spielschlüsseln. Es gibt ein Tool zur Generierung eines Aktivierungscodes:
    Sie können das Tool hier finden: Wir sind nicht mit dem Ersteller des Tools verbunden. Bei Problemen sollten Sie sich direkt an ihn wenden.
  • Schwarze Bildschirme auf Karten, Abstürze mit der Spielkarte, Abstürze beim Erstellen von Screenshots:
    Dieses Problem tritt häufig bei Nvidia-Grafikkarten auf. Wir haben 3 Fixes im Launcher unter "Discovery". Je nach GPU und System funktioniert eine der Lösungen besser für den Nutzer.
    Wichtig: DxVoodoo hat den meisten Nutzern mit RTX-Karten geholfen. Es kommt mit einem Konfigurationstool, das im Einstellungsordner des Launchers zu finden ist. Passen Sie die Einstellungen nach Bedarf an. Aus Erfahrung hilft es, den Arbeitsspeicher zu erhöhen und das DirectX-Level auf das Maximum zu setzen.
    Der Vulkan-Crash-Fix funktioniert besser bei TI-Karten und erfordert keine zusätzlichen Einstellungen. Einfach herunterladen, und der Launcher erledigt den Rest.
    Die Verwendung dieser Fixes löst nicht 100% der Abstürze, verringert jedoch die Häufigkeit. Wir empfehlen auch die Verwendung der 4GB gepatchten EXE aus dem Discovery-Tab.
  • "Verbindung zum Server fehlgeschlagen" Fehler im Spiel:
    Dieser Fehler kann verschiedene Ursachen haben. Stellen Sie sicher, dass a) das Spiel auf dem neuesten Stand ist und b) die Verbindung nicht blockiert wird (falsches Port-Forwarding). Um auf Nummer sicher zu gehen, stellen Sie die Verbindung über den Launcher ein.
    Wenn Sie repackte Spiele verwenden, kann es sein, dass die Spiel-exe online abgelehnt wird. Um Probleme zu vermeiden, wurde die Unterstützung für solche EXE-Dateien deaktiviert.
    Spielername bereits vergeben? Besuchen Sie die Website: Prüfen Sie hier, ob Ihr gewünschter Name verfügbar ist. Wenn der Name bereits vergeben ist, verwenden Sie einen anderen oder wenden Sie sich an ein Teammitglied (Discord, Forum).
  • Andere Multiplayer-Mod-Installation:
    Wenn Sie über ein anderes Mod-Projekt hierher gekommen sind, empfehlen wir eine saubere Installation. Das Spiel-Datei-Update kann auch bei der Installation helfen.
    Die Mods, die Sie vom Mod erhalten haben, sind 1:1 kompatibel. Wichtig: Führen Sie niemals das Spiel-Datei-Update aus, wenn Sie ein repacktes Spiel verwenden, da dies die Installation beschädigen kann!

4. Häufig gestellte Fragen (FAQ)

  • Ist Geld Cheating erlaubt? Im Prinzip ist das uns egal, jedoch sollte man wissen das wir einen Anticheat Implementiert haben der auf Manipulierte Regionen im Memory reagiert und das Spiel beendet
  • Kann ich Mod-Autos verwenden? Ja, solange sie nicht a) im Community Racing Center für offizielle Rennen verwendet werden oder b) andere Spieler stören (Abstürze, etc.).
  • Ist Multiplayer kostenlos? Ja, es gibt keine Gebühr, Spende oder ähnliches für den Online-Zugang. Wenn Sie mit den Serverkosten helfen möchten (natürlich nur, wenn Sie können), kann dies über Patreon erfolgen.
